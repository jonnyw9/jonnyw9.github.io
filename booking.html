Booking System

Description

As part of my final year dissertation project, I decided to build a system which would allow students to book lecturers for tutorial sessions easily and without hassle. This was because it was difficult for students to get time with lecturers.

The system was designed to be easy to use and simple. Students use the system to view their calendar and lecturers use the system view their calendar and manage events.

The system is up and running on the link below. Please feel free to play around and try the web app out. Use the email: test@testuser.com and the password: password123 if you don't wish to use your own information!
