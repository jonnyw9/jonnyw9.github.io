Description

This project was designed to use various different APIs such as Google Maps, Google Geocoder, Twitter, and weather APIs, and then deploy the product in the cloud.

With this module the aim was to demonstrate how to deploy a simple application in a LAMP stack in a cloud virtual machine, as well as using languages like PHP and jquery, to interact with the APIs mentioned above.

So for example, the application would use an API to get the users location then display the weather for that location. Meanwhile the application would get tweets within the United Kingdom and display those tweets at the bottom. Any tweets with location enabled would then display as a marker on the map. The user could then click the marker to see more about the tweet as well as the distance to that tweet from their location using the Google Geocoder API.
