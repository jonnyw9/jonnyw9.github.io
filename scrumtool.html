Description

The Scrum Tool is a group project completed during my final year of university. The project was intended to be a tool which professional teams could manage their Sprints.

The system was split into 5 components, with my component being the component to notifiy users. The system was built with ASP .NET Core and Angular.

For the notification component, I was tasked with making the notifications realtime, so they would pop up when a notification is triggered. I was tasked with making the notifications come through email as well. I also was tasked with allowing users to choose which notifications they recieved.

The realtime notification componenent was made using SignalR, which would listen on the users id for any notifications would would occur, and then a plugin called Toastr would display that notification on the screen.

The email was completed using the Outlook SMTP server, which would send an email to the user when a notification is created.
